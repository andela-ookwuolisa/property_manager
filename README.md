### This is a simple CRUD API
#### To run the file, clone this repo
- clone this repo 
- install the dependencies (pip install requirements.txt)
- create the sql database like so in a python3 environment:
  - `from app create_db`
  - `create_db()`
#### Running the app
  - `flask run`
  
#### Running the test
- `python -m unittest discover -s test`
